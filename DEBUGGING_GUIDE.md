# Flutter Hot Reload, DevTools, and Debugging Guide

## Hot Reload & Hot Restart
- **Hot Reload (Ctrl+S / Cmd+S):**
  - Preserves app state
  - Instantly updates UI for most code changes
  - Use for UI/layout tweaks, minor logic changes
  - Limitations: Does not work for changes in `main()`, `initState`, or native code
- **Hot Restart (Ctrl+Shift+S / Cmd+Shift+S):**
  - Resets app state, rebuilds entire app
  - Use for model/state changes, dependency injection, or when Hot Reload fails

## Flutter DevTools Setup
1. Install DevTools:
   ```sh
   flutter pub global activate devtools
   flutter pub global run devtools
   ```
2. Features:
   - **Widget Inspector:** Visualize widget tree, debug layouts
   - **Timeline:** Profile performance, identify jank
   - **Memory:** Track memory usage and leaks
   - **Network:** Inspect API calls
   - **Logging:** View console output and errors
   - **Debugger:** Set breakpoints, step through code

## VS Code Configuration
- Install **Flutter** and **Dart** extensions
- Enable format on save (Settings > Format On Save)
- Configure `launch.json` for debugging (auto-generated by VS Code)
- Use keyboard shortcuts:
  - Hot Reload: `Ctrl+S`
  - Hot Restart: `Ctrl+Shift+S`
  - Open DevTools: `Ctrl+Shift+P` > "Dart: Open DevTools"

## Android Studio Configuration
- Enable Flutter and Dart plugins
- Configure Dart Analysis and auto-import
- Enable live templates for Flutter widgets
- Enable auto-format on save

## Linting & Formatting
- In `analysis_options.yaml`:
  ```yaml
  include: package:flutter_lints/flutter.yaml
  linter:
    rules:
      - prefer_const_constructors
      - avoid_print
      - prefer_single_quotes
      - sort_child_properties_last
  ```
- Run `flutter format .` to auto-format code

## Debugging Best Practices
- Use **debugPrint()** instead of print()
- Use breakpoints and step-through debugging
- Use Widget Inspector to select and highlight widgets
- Use Performance tab to monitor frame rendering
- Use Memory tab to check for leaks
- Use Network tab to inspect HTTP requests

## Common Issues & Solutions
- Hot Reload not working? Try Hot Restart or full restart
- Widget not updating? Check if setState is called
- Lint warnings? Run `flutter analyze` and fix issues
- DevTools not opening? Run `flutter pub global run devtools` and open the link
